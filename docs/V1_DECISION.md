# V1 Decision: Golem

## Decisión
Golem V1 se considera sustancialmente cerrada.

## Motivo
La evidencia ya reunida muestra que Golem cumple su función mínima real:

- panel del gateway como superficie principal
- WhatsApp como canal auxiliar/remoto
- browser relay funcional por Chrome
- lectura de pestaña abierta
- resumen útil de contenido web
- visibilidad de sesiones en el gateway
- baseline y doctrina ya reflejados en el repo

## Estado
V1 = cerrada con notas.

## Notas
Persisten pequeñas fragilidades operativas en:
- reattach manual del browser relay
- repetibilidad fina de algunas interacciones de sesión

Estas fragilidades no impiden considerar V1 como cerrada.
Se trasladan como observaciones a V1.5 y no bloquean el avance del proyecto.

## Consecuencia
A partir de esta decisión:
- no se expande más V1
- las mejoras nuevas pasan a V1.5
- el foco se mueve a operatividad ampliada sin worker externo
