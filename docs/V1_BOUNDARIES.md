# V1 Boundaries: Golem

## Qué es Golem en V1
Un operador conversacional con acceso al panel, WhatsApp auxiliar y lectura de navegador.

## Qué no es Golem en V1
No es todavía:
- un sistema de desktop agency completa
- un reemplazo de Codex
- un worker autónomo de repo
- un daemon de vigilancia total

## Frontera de seguridad
Las mutaciones estructurales del host y del sistema siguen quedando fuera de Golem V1.
