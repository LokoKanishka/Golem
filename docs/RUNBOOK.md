# Runbook inicial

## Objetivo
Mantener a Golem como sistema gobernable y reproducible.

## Fases
1. Definir contrato OpenClaw <-> Codex.
2. Baselinear el estado vivo de OpenClaw.
3. Crear scripts de deploy/verify.
4. Integrar worker Codex.
5. Integrar notificaciones y entrega de artefactos.

## Prohibiciones
- No parchear todo directamente en ~/.openclaw sin reflejarlo en el repo.
- No usar WhatsApp como sesión ontológica principal.
- No mezclar artefactos con conversación.
