# OpenClaw en Golem

Esta carpeta no es un modulo activo de despliegue.

Su semantica actual es:

- placeholders y scaffolding documental para configuracion/identidad;
- rastro historico del bootstrap inicial del repo;
- lugar reservado para templates o defaults futuros si alguna vez se versionan de forma segura.

Regla:

- no asumir que lo que vive aca gobierna el runtime real de OpenClaw;
- no tocar `~/.openclaw` desde este repo por defecto;
- tratar `openclaw/` como documentacion y estructura de apoyo, no como fuente activa del servicio vivo.
